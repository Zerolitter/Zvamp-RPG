[h1]Zvampext RPG V2.1.0[/h1]

Zvampext RPG is a Killing Floor 2 ServerExt revamp that keeps the classic ServerExt package names for compatibility while adding a custom RPG progression layer, perk UI, admin tools, and server quality-of-life features.

[b]Current state[/b]
[h2][b]Stable for now[/b][/h2]

[h2][b]Latest Unpublished Build is UNAVAILABLE on the test server[/b][/h2]
If 1 is set as password, the server is ahead of release. You may still join.
To join, open the console with [b]F3[/b] and type:
[code]
Open play.aleforge.net:31145
[/code]
Can't join? Delete both folders:
[code]
C:\\Users\\<user>\\Documents\\My Games\\KillingFloor2\\KFGame\\Cache
C:\\Users\\<user>\\Documents\\My Games\\KillingFloor2\\KFGame\\Unpublished
[/code]


[h2]What is included[/h2]
[list]
[*]Custom Zvampext RPG perk progression built on ServerExt and ServerExtMut.
[*]Lobby and in-game Perk V2 UI for perk selection, stat spending, trait configuration, prestige, reset, and unload actions.
[*]Custom Zvampext endless game class for RPG server flow.
[*]Admin dashboard features for pickup tuning, grenade tuning, trader controls, wave progress, server messages, and player management.
[*]Compatibility helpers for custom trader items, ZedSpawner, ZedCosmetic, TF2 Sentry, TraderGuard style trader control, and related server-side workflows.
[*]Localized ServerExt text files and Zvampext trait icon assets.
[/list]

[h2]V2.1.0 highlights[/h2]
[list]
[*]Fixed perk replication readiness when joining through lobby and spawning before all perk data is loaded.
[*]Fixed trait data recovery so in-game menus do not remain stuck on loading perk data.
[*]Fixed trader weapon purchases that could pass invalid upgrade level 255.
[*]Added server title branding and Zvamp presents header text to the spawned Perk UI.
[/list]

[h2]Server setup[/h2]
[code]
Mutator=ServerExtMut.ServerExtMut
Game=ServerExtMut.Zvampext_Endless
[/code]

The package names remain ServerExt and ServerExtMut to preserve compatibility with existing ServerExt-based servers and tools.
